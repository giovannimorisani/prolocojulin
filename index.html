📱 Icona PWA (iPhone)

Nella tua repo GitHub, crea la cartella icons/ e carica le seguenti immagini dal tuo logo:

icons/icon-192.png → 192×192 px
icons/icon-512.png → 512×512 px
icons/icon-180.png → 180×180 px (Apple Touch)
icons/icon-152.png → 152×152 px
icons/icon-120.png → 120×120 px

Strumento gratuito per generare tutte le dimensioni: pwabuilder.com oppure favicon.io

💡 Su iPhone: apri il sito in Safari → Condividi → Aggiungi a schermata Home

⚡ Offline + Service Worker

Crea il file sw.js nella root della repo con questo contenuto per il funzionamento offline:

const CACHE='julin-v1';
const ASSETS=['./','./index.html'];
self.addEventListener('install',e=>e.waitUntil(caches.open(CACHE).then(c=>c.addAll(ASSETS))));
self.addEventListener('fetch',e=>e.respondWith(caches.match(e.request).then(r=>r||fetch(e.request))));
